Dotfiles
========

Requirements:

```
sudo apt-get install exuberant-ctags
brew install ctags-exuberant
```

Setting up:

```
cd ~

git clone git@github.com:dentafrice/dotfiles.git
git@github.com:dentafrice/private_dotfiles.git

cd dotfiles
./bootstrap.sh
```

Updating / pushing dotfiles:

```
update_dotfiles
push_dotfiles
```
